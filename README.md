# Aurora E-Library

A self-contained browser app for managing an e-library collection.

## Features

- Dashboard with collection health, availability, overdue loans, fines, and action queue.
- Catalog with 26 books, functional search, category filtering, availability filtering, sorting, grid/list modes, borrowing, reservations, and digital access links.
- Circulation workflows for active loans, returns, due dates, overdue status, and automatic fine calculation.
- Member registry with plan/status tracking and activity history.
- Admin forms for acquisitions and member registration.
- JSON export/import and demo-data reset.
- IndexedDB persistence with a built-in search benchmark.
- Responsive light/dark interface.

## Run

Open `index.html` in a browser, or serve the folder with any static file server.
